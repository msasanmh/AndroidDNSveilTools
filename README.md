### DNSveil - DNS Tools For Android
**(A Hobby Project)**
- Find and scrap DNS servers from the web, scan and find the fastest DNS servers.
- DNS Lookup Supported Protocols:
  - Anonymized DNSCrypt
  - DNSCrypt
  - DoH (DNS Over HTTPS)
  - DoT (DNS Over TLS)
  - Plain DNS (UDP & TCP)
